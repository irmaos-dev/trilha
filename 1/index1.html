HTML e CSS

Introdução

Bootstrap é um framework web com código-fonte aberto desenvolvido pelo Twitter para desenvolvimento de interfaces de aplicações web usando HTML, CSS e JavaScript.

Um framework é um facilitador no desenvolvimento de uma aplicação, poupando tempo e custos através de uma série de componentes predefinidas, como botões, ícones, tipografias, formulários e layouts responsivos, mas mantendo visual agradável e amigável.

Este projeto visa dar os primeiros passos na construção de interfaces web e na utilização da documentação de uma ferramenta de desenvolvimento.

Relevância profissional

Todas as aplicações web utilizam HTML e CSS. E a maior partes delas utilizam algum tipo de framework CSS.

Dentre as opções listadas no artigo sobre frameworks CSS (https://www.brasilcode.com.br/frameworks-css/), foi verificado que o Bootstrap é o framework com maior número de aparições nas divulgações de vagas para desenvolvedores front-end (https://github.com/frontendbr/vagas/issues?q=is%3Aissue+bootstrap+).

Além disso, os diversos frameworks possuem funcionamento semelhantes, portanto ao aprender Bootstrap, aprender outro framework se torna bem mais fácil.

Finalmente, é bastante comum no contexto profissional que o desenvolvedor receba um arquivo, imagem ou mesmo apenas um esboço contendo um esquema de como o site deve ser. No projeto desse módulo, você receberá um arquivo PDF com a imagem do site a ser desenvolvido por você. Seja criterioso com o resultado.
